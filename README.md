# How to use
- all command
```
$ bootstrap.bat
```

- one command
```
$ bootstrap.bat -c COMMAND
```

And then, please add the created bin folder to the environment variables.
If you need more info, please use -h option.

If you output every command, you may include unnecessary ones, and some commands could conflict with other Windows commands. Therefore, itâ€™s recommended to output only the commands you actually need.
